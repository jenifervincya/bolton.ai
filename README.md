# ⚡ BoltOn.AI

## Plug-and-Play Industrial IoT System for Smart Machine Monitoring and Predictive Maintenance

BoltOn.AI is an Industry 4.0 solution designed to help small and medium-sized manufacturing industries monitor machine health in real time and predict failures before breakdowns occur.

The system uses a non-invasive magnetic sensor node that can be attached to industrial machines without modifying existing equipment. By combining Industrial IoT, Edge AI, FFT-based signal analysis, and machine learning, BoltOn.AI enables affordable predictive maintenance for legacy and heterogeneous machines.

🌐 **Live Demo:** https://jenifervincya.github.io/bolton.ai/

---

## 🚀 Problem Statement

Many SMEs operate legacy industrial machines without built-in monitoring systems. Machine faults are often identified only after failures occur, leading to downtime, productivity loss, and maintenance expenses.

Existing Industry 4.0 solutions typically require expensive sensors, PLC integration, and complex installation procedures, making adoption difficult for smaller industries.

---

## 💡 Proposed Solution

BoltOn.AI introduces a plug-and-play monitoring approach using a magnetic Bolt-Node that can be installed within seconds.

The system:

* Collects vibration, temperature, and current data
* Performs FFT-based signal analysis
* Uses Edge AI anomaly detection
* Predicts potential machine faults
* Streams real-time data through MQTT
* Displays machine health on a live dashboard
